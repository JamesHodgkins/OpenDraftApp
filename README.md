# OpenDraft PySide6 App

This is a basic PySide6 desktop application starter project.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   python main.py
   ```

## Project Structure
- main.py: Main application entry point
- requirements.txt: Python dependencies
- .github/copilot-instructions.md: Project instructions

## Customization
Edit `main.py` to add your own UI and features.
